Decision Maker

What?

Decision Maker is a program that decides for you. It doesn't guarantee the decision is correct. it is just for fun.

How?

It is easy to use program but there is something you should know. There are four levels for decision strength that you have to decide. These are...

  • 0: This decision is the weakest decision. Probably gonna harm you.
  • 1: A bit stronger decision than 0, still weak. Not gonna harm you but you won't get anything if you do that.
  • 2: Stong decision that you get profit if you do it.
  • 3: The strongest decision.

You can add unlimited amount of options to the program.

Any Rules?

There is no rule, you have to obey strength level thing. Almost forgot, you can't add multiple options those have same name.
